# Challenge

## 1. Instrucciones de Construcción

Este proyecto utiliza **Gradle** y **Java 8**

### Clonar el repositorio

```bash
git clone https://github.com/mdgloballogic/challenge.git
cd challenge
```

### Buildear el proyecto

```bash
./gradlew build
```

## 2. Instrucciones de Ejecución

### Levantar el proyecto

```bash
./gradlew bootRun
```
